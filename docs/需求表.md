# CtxForge 需求表：功能边界

| 模块 | 包含范围 | 暂不包含 |
| --- | --- | --- |
| 分层 Memory 系统 | 工作记忆、会话摘要、长期向量记忆的读写与检索 | 完整知识库管理、复杂权限系统、多用户协作 |
| Context Manager | 按固定结构拼装 prompt、控制 token 预算、区分稳定 prefix 与动态内容 | 自动理解任意项目结构、复杂代码索引、全自动 prompt 优化 |
| Prefix-cache 感知管理 | 标记可缓存区域、检测 prefix 变化、输出 cache hit/miss 报告 | 精确账单系统、跨供应商 cache 标准化、底层推理加速 |
| DeepSeek Client | 封装 DeepSeek API 调用、记录请求上下文、支持基础配置 | 多模型路由、复杂重试策略、企业级限流 |
| Skill 系统 | 本地 skill 定义、发现、安装、激活与上下文注入 | 在线 marketplace、评分系统、远程自动更新 |
| 轻量 Agent Demo | 提供 CLI/TUI 入口，演示 memory、context、skill、DeepSeek 调用链路 | 对标成熟编码 Agent、自动大规模改代码、完整 IDE 集成 |
| 项目配置 | 使用配置文件管理模型、memory、skill、上下文预算等参数 | 图形化配置后台、团队级配置同步 |
| 可观测性 | 展示上下文组成、token 占用、cache 变化、memory 命中记录 | 完整 APM、分布式 tracing、复杂成本分析平台 |

## MVP 边界

第一版只需要跑通一条完整链路：

用户输入任务 -> 选择 Skill -> 检索 Memory -> 构建 Context -> 调用 DeepSeek -> 输出回答与 cache 报告 -> 写入会话摘要。
